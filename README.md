# 🌍 RoamRipple

AI-powered travel planner built using HTML, CSS, JavaScript and Firebase.

## 🚀 Live Demo
https://roamripple-7bbef.web.app


## ✨ Features

- User Authentication (Email & Google)
- Trip Booking
- Firestore Database
- AI Travel Guide
- Weather Information
- Google Maps
- Responsive Design
- Toast Notifications
- Loader Animation

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Cloud Firestore
- Open-Meteo API
- Google Maps Embed
